# NeurMHSP: Neural Networks for Multi-Horizon Stochastic Programming

## Overview

**NeurMHSP** is a machine learning–enhanced optimisation framework for solving **Multi-Horizon Stochastic Programming (MHSP)** problems using **neural network surrogate models**.

The core idea is to approximate computationally expensive **operational subproblems** with a trained neural network, which is then **embedded directly into the optimisation model**. This significantly reduces computational time while maintaining high-quality solutions and strong out-of-sample performance.

This repository accompanies the paper:

> Zhang, H., Sormani, G., Messina, E., King, A., Maggioni, F.  
> *Neural Networks for Multi-Horizon Stochastic Programming* (2025)
